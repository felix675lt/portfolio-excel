# 김프 적립 신호 — 로컬 데스크톱 앱

내 컴퓨터에서만 도는 파이썬 GUI. **API 키 불필요**(전부 공개 시세). 유출될 게 없어 안전합니다.

## 실행
- **`실행.command` 더블클릭** (macOS 시스템 파이썬 `/usr/bin/python3` 자동 사용 — tkinter 내장)
- 또는 터미널: `/usr/bin/python3 kimchi_signal.py`
- 데이터·계산만 확인: `/usr/bin/python3 kimchi_signal.py --test`

> ⚠️ homebrew 파이썬(`/opt/homebrew/bin/python3`)은 tkinter가 빠져 있는 경우가 많습니다.
> 런처가 tkinter 있는 파이썬을 자동으로 찾습니다. 정 안 되면 `brew install python-tk`.

## 하는 일 (시세·계산 전용)
- 업비트/바이낸스 실시간 **김프/역프 %** + 신호(강한역프/역프/중립/김프)
- ATH 대비 **드로다운** + 이번 달 **추천 매수액**(바닥선+스트레치)
- **경로 비교**: 업비트 BTC 직접 vs XRP/TRX 다리 → 어느 쪽이 BTC 개수 더 많나
  (개인지갑 출금 체크 시 출금 수수료까지 반영)
- **전송 체크리스트**: 정확한 수량이 찍힌 수동 실행 안내
- ⚙ 내 거래소 수수료 입력 → 계산 반영, 40초 자동 갱신

## 안 하는 일 (의도적으로)
- **자동 매수·자동 전송 안 함.** 실제 주문·송금은 본인이 거래소에서 직접.
- 이유: 출금 권한 API 키를 로컬에 두더라도 자동 송금은 사고(버그·오입력) 시 손실 위험이 큼.
  이 앱은 "무엇을 얼마나" 정확히 알려주고, 실행은 사람이 확인하며 하는 설계.

## 설정 파일
- `kimchi_config.json` — 수수료·옵션 저장 (자동 생성, 이 기기에만)

정보 제공용 · 투자자문 아님 · 프리미엄은 수분 단위로 변동.
